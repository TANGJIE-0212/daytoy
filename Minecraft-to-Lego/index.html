How It Works

from pixels to bricks.

⚡

Features

Built for hardcore
builders.

🗺️

Schematic File Support

Full support for .schematic (pre-1.13), .schem (WorldEdit), and .litematic (Litematica). No data loss, full 3D fidelity.

🎨

Color-Accurate Mapping

All 16 wool, concrete, and terracotta colors. All 16 stained glass variants. Mapped to the closest real LEGO color in Delta-E space.

📐

Brick Optimization

Adjacent same-color 1×1 bricks automatically merged into 1×2, 1×4, 2×4 plates — fewer parts, lower cost, easier build.

🛒

One-Click BrickLink Cart

Export a BrickLink XML wanted list. Instantly find the cheapest sellers for every part in your build, globally.

👁️

Interactive 3D Preview

Rotate, zoom, and inspect your LEGO model in the browser before you order a single brick. Powered by Three.js.

📋

Build Instructions

Layer-by-layer build guide generated automatically. Export as PDF or view in-browser. Build exactly like an official LEGO set.
